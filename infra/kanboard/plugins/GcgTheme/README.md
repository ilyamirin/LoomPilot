# GcgTheme

`GcgTheme` is a lightweight Kanboard theme plugin for this demo repository.

It does three things without forking Kanboard core:

- injects a custom CSS layer
- overrides the login screen composition
- overrides the main header shell

The design direction is based on a golden-canon-inspired editorial grid:
asymmetric proportions, stronger whitespace rhythm, and calmer operator-facing
cards.
